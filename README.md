# :rocket: How to Run

```bash
# Clone Repository
$ git clone https://github.com/LuanSilveiraSouza/apnp-project.git

# Go to project folder
$ cd apnp-project/server

# Install Packages
$ npm i

# Build dist
$ npm build

# Watch changes to build dist
$ npm start
```

Serve the dist folder with Live Server Extension or a Http Server
